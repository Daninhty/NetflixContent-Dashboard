# 🎬 Netflix Content Analytics Dashboard

## 📌 Overview

This project presents an interactive data analytics dashboard that explores Netflix's content library. The analysis focuses on uncovering insights related to content distribution, genre trends, release patterns, and key contributors such as directors.

The dashboard is built using **Power BI**, supported by data cleaning and transformation processes performed in **Microsfot Excel and SQL Server**.

---

## 🎯 Objectives

* Analyze the distribution of Netflix content by genre
* Identify trends in content releases over time
* Compare the proportion of Movies vs TV Shows
* Discover top directors based on the number of titles
* Provide data-driven insights through interactive visualizations

---

## 📊 Key Features

* **Total Content & Directors Overview**
* **Content Growth Trend by Year**
* **Top Genre Distribution (Treemap)**
* **Content Type Breakdown (Movie vs TV Show)**
* **Top 10 Directors by Number of Shows**

---

## 🛠️ Tools & Technologies

* **Microsoft Excel** → Data Cleaning & Preparation
* **SQL Server** → Data Transformation & Querying
* **Power BI** → Data Visualization & Dashboard

---

## 🧹 Data Preparation

Key preprocessing steps include:

* Handling missing values (e.g., "Tidak Tersedia")
* Splitting multi-value columns (e.g., country, genre)
* Creating `genre_group` for simplified analysis
* Removing duplicates and ensuring data consistency

---

## 📈 Key Insights

* Netflix content shows significant growth after 2016
* **Drama** and **International** genres dominate the platform
* Movies account for the majority of content compared to TV Shows
* A small number of directors contribute to a large portion of content

---

## How to Use

1. Download the `.pbix` file from this repository
